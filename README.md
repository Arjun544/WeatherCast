# WeatherCast

### WeatherCast is a web based app, which gives live weather updates of any location.

## Stack:
1. React
2. Tailwind Css
3. WeatherApi

## Features: 
1. Current user location
2. Weather of current location
3. Current day details
4. Weeekly details
5. Per hour details
6. Search location
